Projeto FINAL

Como última atividade, vocês deverão desenvolver um projeto utilizando NodeJS, e o principal requisito desse projeto é a implementação de operações CRUD. Seguem mais detalhes e instruções sobre o trabalho: Presença na Aula: É fundamental que todos compareçam à sala de aula no dia designado para a apresentação. Esta é uma ocasião importante não apenas para mostrar o que vocês fizeram. Apresentação do Projeto: Durante a apresentação, cada dupla terá a responsabilidade de explicar o projeto que desenvolveu. Além de fornecer uma visão geral, é crucial que mostrem e expliquem trechos específicos do código, para que todos possam entender as principais funcionalidades e decisões de design tomadas. Data da Apresentação: Marquem em seus calendários: o projeto deve ser apresentado em 27/11/23. O presente trabalho faz uso, de boodstrap, armazenamento em banco de dados, consulta de api, conecção com banco, apresenta tabela armazenada no banco, apresenta dados de uma api organizada em divs, e validação de dados cadastrados pelo usuário
